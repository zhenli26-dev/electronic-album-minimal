# 四张图片模板说明

## 布局设计
四张图片模板可以采用以下布局之一：

### 布局1：2×2网格
```
┌─────────┬─────────┐
│  图片1  │  图片2  │
├─────────┼─────────┤
│  图片3  │  图片4  │
└─────────┴─────────┘
```
- 四张图片等分空间
- 适合方形或近似方形的图片
- 简洁规整，视觉平衡

### 布局2：1大3小
```
┌───────────────────┐
│                   │
│      大图         │
│                   │
├─────┬─────┬─────┤
│ 小1 │ 小2 │ 小3 │
└─────┴─────┴─────┘
```
- 上方一张大图突出展示
- 下方三张小图并排
- 适合有主次之分的图片组

### 布局3：3小1大
```
┌─────┬─────┬─────┐
│ 小1 │ 小2 │ 小3 │
├─────┴─────┴─────┤
│                 │
│      大图       │
│                 │
└─────────────────┘
```
- 上方三张小图并排
- 下方一张大图
- 适合渐进式展示

## 使用建议
1. **图片选择**：选择主题相关、视觉风格一致的图片
2. **顺序安排**：按时间顺序、重要性或视觉逻辑排列
3. **间距控制**：图片之间保持适当间距，通常10-20像素
4. **边框处理**：可添加细边框或阴影增强视觉效果
5. **文字说明**：可在图片下方添加简短说明文字
